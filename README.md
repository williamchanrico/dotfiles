# Dotfiles
Config files running gnome on xorg-server

## YADM Setup
```
#!/usr/bin/env bash

# Set up YADM
yay -S --noconfirm yadm-git
yadm clone https://bitbucket.org/williamchanrico/dotfiles
yadm decrypt
```
