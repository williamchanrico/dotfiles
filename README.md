# linux-config-files
Config files for archlinux running gnome on xorg

## Dotfiles setup
```
#!/usr/bin/env bash

# Set up YADM and dotfiles
yay -S --noconfirm yadm-git
yadm clone https://bitbucket.org/williamchanrico/dotfiles
yadm decrypt
```
